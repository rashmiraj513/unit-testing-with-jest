# Unit Testing for Typescript & NodeJs Developers with Jest

This repository covers the unit testing in a TypeScript and Node.js environment using Jest. It includes examples and exercises to help you understand how to write and run tests effectively.

## Getting Started

### Prerequisites

- Node.js
- npm (Node Package Manager)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/rashmiraj513/unit-testing-with-jest.git
   ```
2. Navigate to the project directory you want to execute:
   ```bash
   cd unit-testing-with-typescript/00_basics-of-testing-with-jest
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

### Running Tests

To run the tests, use the following command:

```sh
npm test
```
